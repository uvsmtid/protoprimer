
The execution mode (`exec_mode`) specifies the following:

*   `mode_cli`

    This means that current `python` process extends `protoprimer` DAG to
    manage CLI (e.g. parses `sys.argv`), logging, and other things.

    Normally, this is the mode run via `EntryFunc.func_boot_env` [entry_func][FT_25_62_13_55.entry_func.md].
