## Intro

Exec mode refers to what [FT_90_65_67_62.proto_code.md][FT_90_65_67_62.proto_code.md] is invoked for.

It changes how the script traverses the [FT_68_54_41_96.state_dependency.md][FT_68_54_41_96.state_dependency.md]
and affects how each state is bootstrapped.

Some of the exec modes can be selected via CLI:

```sh
./prime -h
```
