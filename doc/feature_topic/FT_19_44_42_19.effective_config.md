## Effective configuration

To make it easily comprehensible, the `protoprimer` supports `eval` sub command to print effective config
where all fields are shown with their evaluated values (if present) including final derived ones.

It is possible to evaluate the effective config to see:
*   the data **loaded** from files
*   the data **derived** from the **loaded** data

```sh
./prime eval
```
