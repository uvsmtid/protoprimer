# FT_08_92_69_92.env_var

With no args (convenience requirement of a bootstrap script),
the only way to override defaults before it reaches config files is to use env vars.

They can also be configured in [shebang_line][FT_02_89_37_65.shebang_line.md].

## Supported env vars

See `EnvVar` enum inside `protoprimer.primer_kernel`.

*   `PROTOPRIMER_STDERR_LOG_LEVEL`

    It holds a `str` value for the constant named as defined by `logging` module.

    For example, value `INFO` refers to `loggint.INFO`.
