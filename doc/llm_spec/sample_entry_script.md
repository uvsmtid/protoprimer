
Script `./cmd/gen_next_doc_id` delegates execution to
function `gen_next_doc_id` inside
module `local_doc.cmd_gen_next_doc_id` implemented
in `./src/local_doc/main/local_doc/cmd_gen_next_doc_id.py` file.

Normally,
`./cmd/gen_next_doc_id` (inside `./cmd` dir but without `cmd_` prefix in extension-less file)
is implemented by
`cmd_gen_next_doc_id.py` (inside one of the projects, with `cmd_` prefix in `*.py` file).
