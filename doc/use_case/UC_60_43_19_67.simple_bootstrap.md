# UC_60_43_19_67.simple_bootstrap

A dev-author should always be able to configure a bootstrap command for dev-users that is both:
*   arg-less
*   one-liner

This is the primary feature of `protoprimer`.

Terminology:
*   A dev-author is the person who maintains the repo and uses `protoprimer` to make life easy for dev-users.
*   A dev-user is the person who mostly clones the repo to run software directly from it (potentially contributing).
